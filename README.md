# HideHelmet

What it does
- Hide armor visuals per-player using a bitmask (self-only).

Commands
- /hidehelmet (toggle head)
- /hidearmor [status|all|on|off] [head|chest|hands|legs]

Note
- Solo afecta tu vista (self-only). Otros jugadores siguen viendo tu armadura normal.

Persistence
- players.json is stored in the plugin data directory.
